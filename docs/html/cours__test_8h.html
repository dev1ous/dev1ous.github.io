int factorial ( int n)

Calculates the factorial of a number.

Parameters
nThe non-negative integer for which to calculate the factorial
Returns
int The factorial of n
Exceptions
std::overflow_errorIf the result exceeds the maximum value of int
Note
Time complexity: O(n)
Warning
Input must be non-negative

The factorial of a non-negative integer n (written as n!) is the product of all positive integers less than or equal to n. For example:

  • 5! = 5 * 4 * 3 * 2 * 1 = 120
  • 0! is defined to be 1

The function uses a recursive implementation.

int result = factorial(5); // Returns 120
int zero_factorial = factorial(0); // Returns 1
